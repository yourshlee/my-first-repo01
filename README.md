# my-first-repo

이 저장소는 git을 연습하기 위해 만든 것입니다.

See Demo [Here](https://www.google.com/)

## Prerequisites

- Python^3.13
- git

## How to start

```shell
$ git clone addr
$ cd my-first-repo
$ python main.py
```

## Installation

There is no installation to execute this project.


## Features

```python
for _ in range(3):
    print('hello')
```

만약 문장 안에서 특정한 코드 ( 예: `print('hello')`) 하이라이트가 필요하다면, backquote로 감싸주시면 됩니다.
